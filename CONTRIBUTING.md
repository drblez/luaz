# Contributing to Lua/TSO

Thanks for your interest in contributing!

## How to contribute

- Open an issue to discuss bugs, ideas, or proposals.
- Submit a pull request with a clear description of the change.
- Keep changes focused and aligned with the project goals in the RFC.

## Development notes

- The project targets z/OS and focuses on dataset-based workflows.
- Keep documentation updates in `docs/`.
