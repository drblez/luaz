# LUZ Code Ranges

This document reserves LUZ ranges by subsystem.

- 00000–09999: Core and shared infrastructure
- 10000–19999: Lua module stubs (Lua layer)
- 20000–29999: Tooling/scripts
- 30000–39999: C core stubs and host APIs
- 40000–49999: Dataset I/O and DDNAME services
- 50000–59999: ISPF services
- 60000–69999: TSO services
- 70000–79999: AXR services
- 80000–89999: TLS/System SSL
